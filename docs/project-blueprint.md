# TerraSight — 프로젝트 구상안

## 한 줄 정의

**전지구 환경 데이터를 실시간 3D 지구본으로 시각화하는 포털.**
기후 시각화로 방문자를 유입 → 환경 데이터 아틀라스로 체류 → 미국 도시별 환경 리포트로 수익(AdSense/SEO).

## Globe 데이터 레이어 설계

### 레이어 분류

| 분류 | 렌더링 방식 | 예시 |
|------|-----------|------|
| **연속 표면** | 전지구 래스터 이미지를 구면에 드레이프 | SST, 대기질, 기온 |
| **이벤트 포인트** | 개별 좌표를 점/아이콘으로 표시 | 산불, 지진 |
| **흐름 애니메이션** | 파티클이 벡터장을 따라 이동 | 해류, 바람 |

### Globe 인터랙션

| 기능 | 설명 |
|------|------|
| **카테고리 탭** | 하단 바에서 레이어 전환 (한 번에 1개 표면 + 1개 이벤트) |
| **클릭 조회** | 아무 지점 클릭 → 해당 좌표의 실제 데이터 값 표시 |
| **호버 상세** | 이벤트 포인트 위에 마우스 → 상세 팝업 |
| **자동 회전** | 8초 미조작 시 지구본 천천히 회전 |
| **3D/2D 전환** | Globe ↔ 평면 지도 morph 애니메이션 |
| **줌/패닝** | 마우스 드래그/스크롤로 탐색 |

## 데이터 소스 전략

### 신뢰도 계층

```
1순위: NASA GIBS (무인증, 무제한, land mask 내장, 안정)
       → SST, AOD, CO₂, Floods, Night Lights, Clouds

2순위: 정부 무인증 API (USGS, NOAA ERDDAP, FIRMS)
       → 지진, 해수면온도(포인트), 산불

3순위: Open-Meteo (무인증, rate limit 있음)
       → 기온, 강수, 대기질, 해류 — 캐시 필수

4순위: Copernicus CDS/ADS (무료 계정 필요)
       → ERA5, CAMS — 원천 격자 데이터
```

### Trust Tag 체계

| 태그 | 색상 | 의미 |
|------|------|------|
| observed | 🟢 | 직접 관측 (계기 측정) |
| near-real-time | 🟡 | 수 시간 내 처리 |
| forecast | 🟠 | 모델 예측 출력 |
| derived | 🔵 | 관측값에서 계산 |
| compliance | ⚪ | 규제 보고 데이터 |

## 기술 스택

### Frontend
- **React 18** + Vite + TypeScript
- **CesiumJS** — 3D 지구본 렌더링 (deck.gl에서 마이그레이션)
- CSS-in-JS (인라인 스타일)
- 다크 테마 (글래스모피즘 UI)

### Backend
- **FastAPI** (Python)
- 커넥터 패턴: 데이터 소스당 1개 클래스 (`backend/connectors/`)
- 렌더링 엔진: numpy + scipy + matplotlib + Pillow
- 파일 캐시: `/tmp/terrasight_cache/` (TTL 기반)

### 배포
- Frontend: **Cloudflare Pages**
- Backend: **Render** (Free Tier, 512MB RAM)
- DNS: Cloudflare

### 외부 서비스
- Cesium Ion: Globe 기본 위성 이미지
- NASA GIBS: 환경 래스터 타일
- NASA FIRMS: 산불 데이터
- USGS: 지진 데이터
- NOAA ERDDAP: SST 격자 데이터
- Open-Meteo: 기상/대기질 격자 데이터

## 차별화 포인트

### vs 경쟁자

| 경쟁자 | 그들의 강점 | TerraSight의 틈새 |
|--------|-----------|-------------------|
| earth.nullschool | 아름다운 대기 시각화 | 수질/토양/시설/규제 데이터 없음 |
| NASA Worldview | 1,200+ 레이어 | 전문가 도구 — 교육/로컬 리포트 없음 |
| Windy.com | 대중 UX, 날씨 | 환경 데이터 없음 |
| EPA/USGS/NOAA | 도메인별 깊이 | 완전히 사일로 — 교차 도메인 스토리 없음 |

### TerraSight만의 3가지

1. **환경공학 교과 분류 체계** — 8개 카테고리가 교과서와 일치
2. **명시적 신뢰도 태그** — 모든 데이터에 observed/forecast/derived 표시
3. **규제↔관측 교차 연결** — EPA 규제 데이터와 위성 관측 데이터를 같은 화면에

## 대상 사용자

### 1차: 환경공학 학생/실무자
- 교과서 카테고리 구조로 데이터 탐색
- Trust tag으로 데이터 신뢰도 즉시 파악
- Cross-domain 연결로 전체 그림 이해

### 2차: 기후 관심 일반인
- Globe의 시각적 임팩트로 유입
- "내 도시의 환경은?" → Local Report
- Born-in Interactive → 소셜 공유

### 3차: 미국 도시 환경 검색자
- "houston air quality" → SEO 유입
- 14개 블록의 포괄적 환경 프로파일
- 관련 도시 추천으로 추가 탐색
